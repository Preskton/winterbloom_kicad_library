A basic project template for Winterbloom. This sets up some useful defaults and common configuration.

  • Page size is US Letter
  • Schematic fields for Rating and MPN
  • Board design rules set up for MacroFab and PCBWay
  • Netclasses for low power, high power, and analog traces
  • Template for PCB specifications
